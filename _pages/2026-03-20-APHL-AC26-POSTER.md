<h1><b><span style='color:black'>Supplementary Methods</span></b></h1>

<h2><b><span style='color:windowtext'>QC and Genome Assembly</span></b></h2>

<p class=MsoNormal>We ran all reads through the CDC/PHoeNIx pipeline, version
2.1.1, to obtain quality control metrics and genome assemblies for each CRAB
isolate. </p>

<p class=MsoNormal><b>&nbsp;</b></p>

<h2><b><span style='color:windowtext'>PopPUNK </span></b></h2>

<p class=MsoNormal>Database Creation</p>

<p class=MsoListParagraphCxSpFirst style='text-indent:-.25in'>1.<span
style='font:7.0pt "Times New Roman"'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; </span>Created
a custom database of 645 CRAB isolates using poppunk Â–create-db.</p>

<p class=MsoListParagraphCxSpLast style='text-indent:-.25in'>2.<span
style='font:7.0pt "Times New Roman"'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; </span>Fit
a Bayesian Gaussian Mixture Model to the database using poppunk Â–fit-model
bgmm.  </p>

<p class=MsoNormal>-PopPIPE</p>

<p class=MsoListParagraphCxSpFirst style='text-indent:-.25in'>1.<span
style='font:7.0pt "Times New Roman"'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; </span>Once
the database was created, we queried with another 29 CRAB isolates using
poppunk_assign.</p>

<p class=MsoListParagraphCxSpLast style='text-indent:-.25in'>2.<span
style='font:7.0pt "Times New Roman"'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; </span>Created
visualizations using poppunk_visualise with the Â–microreact flag.</p>

<p class=MsoNormal>+PopPIPE </p>

<p class=MsoListParagraphCxSpFirst style='text-indent:-.25in'>1.<span
style='font:7.0pt "Times New Roman"'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; </span>Once
the database was created, we ran the PopPIPE workflow on the database. </p>

<p class=MsoListParagraphCxSpMiddle style='text-indent:-.25in'>2.<span
style='font:7.0pt "Times New Roman"'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; </span>Queried
the database with another 29 CRAB isolates using poppunk_assign. </p>

<p class=MsoListParagraphCxSpLast style='text-indent:-.25in'>3.<span
style='font:7.0pt "Times New Roman"'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; </span>Ran
PopPIPE workflow to get subcluster identities for the 29 CRAB isolates.</p>

<h2><b><span style='color:windowtext'>floc</span></b></h2>

<p class=MsoListParagraphCxSpFirst style='text-indent:-.25in'>1.<span
style='font:7.0pt "Times New Roman"'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; </span>Sketched
645 CRAB isolates to create a database. Used kmer size of 31 and scaled value
of 100</p>

<p class=MsoListParagraphCxSpMiddle style='text-indent:-.25in'>2.<span
style='font:7.0pt "Times New Roman"'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; </span>Queried
the database with another 29 CRAB isolates.</p>

<p class=MsoListParagraphCxSpLast style='text-indent:-.25in'>3.<span
style='font:7.0pt "Times New Roman"'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; </span>Ran distance
output through custom python script (using BioPython Phylo) to obtain neighbor-joining
trees.</p>

<p class=MsoNormal>&nbsp;</p>

<h2><b><span style='color:windowtext'>Dryad</span></b></h2>

<p class=MsoNormal>Once we had clustering outputs from PopPUNK and floc, we
noticed that floc cluster 5 (flocÂ’s largest cluster output) and PopPUNK+PopPIPE
strain 2 contained similar DCLS cluster IDs. We decided to run genome
assemblies belonging to floc cluster 5 and PopPUNK strain 2 through Dryad, a
DCLS validated reference-based SNP calling pipeline. We wanted to see the SNP
ranges of cluster 5 and strain 2, as well as the tree topology on Dryad trees
compared to PopPUNK+PopPIPE and floc trees. </p>

<h2><span style='color:windowtext'>&nbsp;</span></h2>

<h2><b><span style='color:windowtext'>Sensitivity and Specificity</span></b></h2>

<p class=MsoNormal>Sensitivity was calculated as (true positives) / (true
positives + false negatives).</p>

<p class=MsoNormal>Specificity was calculated as (true negatives) / (true
negatives + false positives)</p>

<p class=MsoNormal>True positives: isolates belonging to the same DCLS cluster
being assigned to the same floc/PopPUNK cluster. </p>

<p class=MsoNormal>False negatives: isolates belonging to the same DCLS cluster
being assigned to different floc/PopPUNK clusters. </p>

<p class=MsoNormal>True negatives: isolates belonging to different DCLS
clusters being assigned to different floc/PopPUNK clusters. </p>

<p class=MsoNormal>False positives: isolates belonging to different DCLS
clusters being assigned to the same floc/PopPUNK cluster.</p>
